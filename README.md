## 📌 Project Overview

This project focuses on analyzing loan application data to identify the key factors influencing loan approvals and predicting whether a loan application is likely to be approved. The project combines SQL, Python, Machine Learning, and Power BI to perform end-to-end data analysis, predictive modeling, and interactive dashboard development.

The objective is to assist financial institutions in making data-driven lending decisions by uncovering customer patterns, evaluating credit risk, and visualizing business insights.

---

## 🎯 Objectives

- Analyze historical loan application data.
- Identify the factors affecting loan approval.
- Perform Exploratory Data Analysis (EDA).
- Build a predictive machine learning model.
- Create an interactive Power BI dashboard.
- Generate actionable business insights for lending decisions.

## 📌 Key Business Insights

- Applicants with good credit history have significantly higher approval rates.
- Medium-income applicants account for the highest number of loan approvals.
- Semiurban areas show better approval performance than rural areas.
- Graduate applicants generally experience higher approval rates.
- Applicant income and credit history are among the strongest predictors of loan approval.
